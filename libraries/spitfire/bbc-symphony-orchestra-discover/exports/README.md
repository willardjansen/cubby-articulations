# Exported Expression Maps

This folder contains DAW-specific expression map files generated from the source YAML files.

## Installation

### Cubase / Nuendo

Copy `.expressionmap` files to:
```
Windows: %APPDATA%\Steinberg\Cubase [version]\Expression Maps\
macOS: ~/Library/Preferences/Cubase [version]/Expression Maps/
```

### Logic Pro

Copy `.plist` files to:
```
~/Music/Audio Music Apps/Articulation Settings/
```

### Studio One

Copy `.keyswitch` files to:
```
~/Documents/Studio One/Sound Variations/
```

## Note

If this folder is empty, the expression maps may need to be generated from the source YAML files using the conversion tools in `/tools`.
